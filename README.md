# rotary-stevenage-grange website. 
Official website redesign for the Rotary Club of Stevenage Grange, Stevenage
**Live Site:** https://sgrc.org.uk
## About this project 
I am the Webmaster of the Rotary Club of Stevenage Grange. This repository 
documents the custom development work I am doing to redesign and improve the club's website.
The site is built on WordPress with Elementor. This repo tracks all custom CSS, JavaScript, and other code I am writing to improve the site's current design, performance, and functionality that goes beyond what the theme and page builder currently provide out of the box. 

## What I am building
1. Full website redesign from an outdated site into a modern one. mobile-first experience
2. Six main navigation sections with eighteen subpages covering the club's full range of activities.
3. photo gallery to showcase the club's projects and events
4. Custom brand styling using the Club's official colour palette
3. Mobile responsiveness improvements
4. A working contact form with JScript validation
5. An interactive events section
6. performance and accessibility improvements
7. Integration with Rotary International member portal
